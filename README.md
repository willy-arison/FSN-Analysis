#### FSN Analysis -  Fast/Slow/Non-Moving Classification

This project provides helper function to compute stock coverage and classify products using FSN analysis. FSN helps businesses understand how products move over time and support better decision in supply chain, inventory, and marketing.

**FSN analysis helps teams answer business questions:**

* Which products sell fast and need stable replenishment? Fast moving items support high revenue and must always remain in stock.
* Which products are slow and require marketing actions?
* Which products are not moving at all?
